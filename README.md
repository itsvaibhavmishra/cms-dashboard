## Installation

### Move to the root directory using command

```bash
$ cd cms-dashboard/
```

### Install dependencies

```bash
$ npm install
```

### Start app on localhost(React App)

```bash
$ npm start
```

### Create an optimized production build

```bash
$ npm run build
```
